Halo!

Silakan letakkan foto-foto kucing Anda di folder ini.
Untuk memudahkan, pastikan nama filenya sesuai dengan nama kucing dan menggunakan huruf kecil, contoh:
- haru.jpg
- hiro.jpg
- jojo.jpg
- hani.jpg
- blacky.jpg
- lolly.jpg
- cherry.jpg
- panda.jpg
- oliver.jpg

Jika format foto Anda bukan .jpg (misalnya .png atau .jpeg), Anda perlu menyesuaikan ekstensi filenya di dalam file `index.html` dan `script.js`.
Selamat mencoba!
